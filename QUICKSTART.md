# 🚀 Quick Start: AI Assistant in 5 Steps

> ⚡ **TL;DR Version** - Full guide: [SETUP-GUIDE.md](SETUP-GUIDE.md)

> ⏱️ **Total Time:** 18-22 minutes (updated with port step!)

---

## ① Get Alibaba API Key (3 min)

1. Sign up: [Alibaba Cloud](https://www.alibabacloud.com/)
2. Go to: **Model Studio** → **API Key**
3. Click: **Create API Key**
4. **Copy & Save** (starts with `sk-`)

🔑 **Free:** 70 million tokens for new users!

---

## ② Create Telegram Bot (2 min)

1. Open Telegram → Search `@BotFather`
2. Send: `/newbot`
3. Name your bot (e.g., `My AI Assistant`)
4. Username must end in `bot` (e.g., `my_ai_helper_bot`)
5. **Copy & Save** the token

🤖 **Token looks like:** `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`

---

## ③ Deploy Server (5 min)

1. Go to: [Simple Application Server](https://swas.console.aliyun.com/)
2. Click: **Create Server**
3. Image: **Application Image** → **OpenClaw**
4. Config: **2 GiB RAM** minimum
5. Region: **US (Virginia)** or **Singapore**
6. Complete purchase (may be FREE!)

☁️ **Wait 2-5 min** for server to start

---

## ④ Configure (5 min)

1. Open server dashboard
2. Click: **Application Details** → **Execute Command**
3. Terminal opens → Follow prompts:
   - Paste **Alibaba API Key**
   - Select **Telegram** channel
   - Paste **Bot Token**
4. Wait for **"Setup Complete!"**

⚙️ **Done!** Your bot is alive!

---

## ⑤ Open Port (2 min)

1. Still in **Application Details**
2. Click: **Execute Command** under **"Open Port"**
3. Firewall auto-configures
4. Wait for **"Port opened successfully!"**

🔓 **Critical!** Telegram needs this to reach your bot!

---

## ⑥ Test (1 min)

1. Open Telegram
2. Search your bot username
3. Send: `/start`
4. Bot responds! 🎉

---

Your AI assistant is now running 24/7!
